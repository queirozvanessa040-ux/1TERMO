## 📚 Repositório de Aulas e Materiais Didáticos
Este repositório contém os materiais de suporte, roteiros de aula e códigos práticos desenvolvidos para as disciplinas de tecnologia, cobrindo desde os fundamentos da programação até a arquitetura de sistemas complexos.
## 🛠️ Disciplinas e Conteúdos Programáticos## 1. Lógica de Programação & Versionamento
Fundamentos de desenvolvimento de software e controle de versão para iniciantes.

* Fundamentos: Algoritmos, variáveis, operadores de dados e estruturas condicionais/repetição.
* Abordagem Multi-Linguagem: Implementação de conceitos em Python, Java e JavaScript.
* Controle de Versão: Inicialização, commits e sincronização de repositórios através do Git e GitHub.

## 2. Arquitetura IoT (Internet das Coisas)
A integração entre o mundo físico e o processamento de dados em nuvem.

* Hardware & Firmware: Ecossistema Arduino e programação de microcontroladores em C++.
* Coleta e Análise: Uso de Python (PySerial) para captura de dados de sensores via comunicação serial.
* Camadas IoT: Estudo das camadas de dispositivo, rede e aplicação.

## 3. Sistemas Operacionais (Fundamentos e Gestão)
Exploração interna do gerenciamento de hardware e metodologias de desenvolvimento de sistemas.

* Arquitetura de Kernels: Análise comparativa profunda entre sistemas Windows (híbrido) e Linux (monolítico).
* Gerenciamento do Sistema: Administração de processos, threads, memória virtual e sistemas de arquivos (NTFS vs. EXT4).
* Ambiente Embarcado: Conceitos de programação bare-metal, super-loops e gerenciamento de concorrência sem bloqueio de CPU (millis()).

## 4. Engenharia de Requisitos & Metodologias Ágeis
Processo de concepção de software aplicados a projetos de infraestrutura de sistemas.

* Levantamento de Escopo: Dinâmicas de brainstorming e entrevistas técnicas com usuários.
* Especificação: Classificação e documentação de Requisitos Funcionais e Não Funcionais.
* Validação: Modelagem através de diagramas técnicos, relatórios de benchmark e prototipagem lógica.
* Gestão Ágil: Execução de projetos utilizando os frameworks Scrum (Sprints, Daily, Backlog) e Kanban (visualização de fluxo e limite de WIP).
